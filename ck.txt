多仓库
当前仓库配置信息：
$ git remote -v
添加仓库 指定代码仓库的别名，此处我使用 gttis 作为我司代码仓库的别名
$ git remote add origin git@gitee.com:liu-yu-mitumao/ceshi.git
添加完后查看仓库名称及地址
$ git remote -v
可以看到我司的代码仓库已经成功添加了，但是此时你是没办法签出新仓库的代码分支的，因为它的仓库信息还没有被拉取下来。
所以我们需要通过fetch命令拉取新仓库的元数据信息（分支、标签等）
$ git fetch gttis
提交代码到本地缓冲区
$ git commit -a -m 'for testing'

首先，将代码推到原代码仓库（gitee.com）
$ git push
紧接着，将代码提交到我司代码仓库
$ git push gttis

克隆目录、下载
git clone git@gitee.com:liu-yu-mitumao/ceshi.git
